Doctor Finder

Find doctors in Portland, 05.26.17

By Ben Petrosky

Description

Allows users to search for doctors in the Portland area based on their symptom.

Setup/Installation Requirements

node most be installed on your computer.

Clone this repository.
in terminal within the project folder :
  npm init
  npm install
  bower init
  bower install
  gulp serve
